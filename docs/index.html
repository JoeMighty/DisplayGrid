Features

Everything you need to run your screens

Built for operators who need reliability — not another SaaS dashboard with a monthly bill.

Multi-zone layouts

Split any screen into independently controlled regions, each running its own playlist simultaneously. Define regions by percentage coordinates — perfect for menu boards with a live footer, or a lobby wall mixing announcements with video.

Real-time delivery

WebSocket pushes playlist changes to every display instantly — no polling, no reloads.

Playlist builder

Drag-and-drop slides with per-slide durations, transitions, and scheduling.

Asset library

Images, videos, and PDFs with automatic WebP optimisation via Sharp.

Screen health

Live online/offline status, last-seen timestamps, and client IPs at a glance.

Emergency override

Broadcast a full-screen urgent message to every display with one click.

Kiosk lock

PIN-protected overlay prevents tampering; unlock with a configurable key combo.

Live streams New

Play HLS (.m3u8) or WebRTC (WHEP) sources as slides. Pair with go2rtc or MediaMTX to put IP cameras and OBS feeds on any screen — the foundation for NDI support on the roadmap.

Time-based scheduling

Restrict slides to specific days and time windows — re-evaluated every minute, automatically.

Offline resilience

Displays cache their last playlist and keep playing through network outages.

Role-based access

Super Admin, Admin, Operator, and Viewer roles with granular permissions.

Multi-screen management

Resolution, refresh rate, rotation, colour profile, and LED panel grids per screen.

Auto-updates New

Server and kiosk apps keep themselves current from GitHub Releases — kiosks update silently on restart.

Docker New

One container runs everything: docker compose up -d, state in a single volume.

Standalone apps

Native installers for Windows, macOS (Universal), and Linux. The server lives in your tray; the kiosk app boots displays fullscreen with zero browser setup.

Raspberry Pi friendly

Run displays on a £35 Pi with Chromium in kiosk mode — server updates reach every Pi instantly, no device maintenance.

On the roadmap

NDI source support

Stream live video from NDI devices on the local network to any display region — building on the live-stream foundation. See the roadmap →

How it works

Up and running in minutes

No cloud account. No credit card. Download an app — or run one Docker command — and follow the wizard.

Install the Server App

Download for Windows, macOS, or Linux — or run docker compose up -d. The database sets itself up on first launch.

Run the setup wizard

The dashboard opens automatically. Name your app, create the Super Admin account, and set a kiosk PIN.

Add zones & screens

Group displays into zones (lobby, corridor, reception), then add screens. Each screen gets a unique pairing token.

Build playlists

Upload images, videos, and PDFs, then arrange slides with custom durations, transitions, and schedules.

Pair your displays

Install the Kiosk App on each display device — or open the display client in any browser — and enter the screen token. It connects and starts playing.

Manage from anywhere

Update playlists from the dashboard; changes are pushed to every paired display in real time via WebSocket.

Go further

Split screens into multi-zone layouts, schedule slides by day and time, and drop live camera or OBS feeds onto any screen.

Stay current, automatically

The apps keep themselves updated from GitHub Releases. Kiosks apply updates silently on restart — never mid-show.

Access control

The right access for every team member

From the IT admin who manages everything to the volunteer who uploads the weekly schedule.

★ Super Admin

Full control

Manage everything: users, screens, content, and settings. Can assign any role, including Super Admin.

Admin

Content & users

Manage all screens, playlists, assets, and user accounts. Cannot delete users or assign Super Admin.

Operator

Content only

Day-to-day content management: screens, zones, playlists, assets. No access to user management.

Viewer

Read only

View all content pages without being able to create, edit, or delete anything.
